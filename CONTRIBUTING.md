# Contributing to Awesome iOS Apps

First off, thank you for considering contributing to the Awesome iOS Apps directory! It's people like you that make this list a great resource for everyone.

## How to Contribute

There are two ways you can contribute to this project:

### 1. Suggest an App (No Coding Required)
If you know a great app but don't want to mess with Git or Pull Requests, you can simply open an issue!
- Go to the **Issues** tab.
- Click **New Issue**.
- Select **Suggest an App** and fill out the form provided.

### 2. Submit a Pull Request (For GitHub Users)
If you're comfortable with Git, submitting a PR is the fastest way to get your app added.

#### Step-by-Step Guide
1. **Fork** the repository to your own GitHub account.
2. **Clone** the project to your local machine.
3. **Create a branch** for your addition (`git checkout -b add-cool-app`).
4. **Update `README.md`**: Find the most appropriate category and add your app using the standard format.
5. **Commit & Push** your changes to your fork.
6. **Open a Pull Request** against the `main` branch of this repository.

## App Submission Guidelines

To keep this directory high-quality and safe, please ensure your submission meets these criteria:

- ✅ **Official App Store Only**: All links must point to the official Apple App Store (`apps.apple.com`). We do not accept third-party sideloading links (e.g., `.ipa` files from unknown sources).
- ✅ **Free or Freemium**: The app must be free to download and use. If it has premium features, the core functionality must still be usable for free.
- ✅ **High Quality**: The app should be relatively stable, well-designed, and useful.
- ✅ **Alphabetical Order**: Please try to insert your app in alphabetical order within its category.

## Standard App Format

Please use the following format when adding an app to the `README.md`:

```markdown
- [App Name](https://apps.apple.com/...) - Brief, 1-2 sentence description of what the app does and why it's great. *(Free / Freemium / In-App Purchases)*
```

**Example:**
- [Tubi](https://apps.apple.com/us/app/tubi-movies-live-tv/id886412752) - A massive legal on-demand library for movies and TV shows. *(Free with Ads)*

Thank you for contributing!
